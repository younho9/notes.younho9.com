- 현재 떠 있는 탭 내에서 데이터를 유지한다.
- 새로고침, 복원(닫은 탭 다시 열기) 시에도 데이터를 유지한다.
- 하지만 다른 탭에는 데이터가 존재하지 않는다.
- [[쿠키(cookie)|쿠키]]와의 차이점
	- [[쿠키(cookie)|쿠키]]는 [[도메인(Domain)|도메인]]을 지정하여 저장하기 때문에, 다른 탭과 [[쿠키(cookie)|쿠키]]가 동기화 된다.
	- 브라우저의 탭이 닫히더라도, 브라우저가 종료되지 않으면, 새 탭에서 쿠키가 유지된다.
