# {{ $frontmatter.title }}

> 입력 : DOM 트리 및 외부 스타일시트 파일
> 출력 : 렌더 트리

- 스타일 시트(CSS)를 파싱하여 CSSOM 트리를 만든다.
- DOM 트리와 CSSOM 트리로 렌더 트리로 구성한다.
- 스타일시트를 파스하여 DOM 노드에 대한 **계산된 스타일**(개발자 도구의 Computed 탭에서 볼 수 있음)을 결정한다.
- 브라우저 마다 갖고 있는 기본 스타일시트 역시 포함된다.
