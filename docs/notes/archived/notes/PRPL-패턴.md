- Preload(또는 Push라고도 함) : 중요한 자원들을 먼저 가져옴.
- Render : 가능한 빠르게 렌더링
- Pre-cache : 나머지 리소스들을 캐시함.
- Lazy load : 다른 경로나 중요하지 않은 리소스를 나중에 로드함.
