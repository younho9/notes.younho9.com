# Vue Composition API

컴포지션 API는 options를 선언하는 것 대신 함수를 import 하여 Vue 컴포넌트를 작성할 수 있도록 하는 API 세트로 아래의 API들을 포함한다.

- Reactive API : `ref()`, `reactive()`
- Lifecycle Hooks : `onMounted()`, `onUnmounted()`
- Dependency Injection : `provide()`, `inject()`
