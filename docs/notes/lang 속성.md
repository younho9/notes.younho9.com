- 웹페이지에 설정된 언어를 선언하는 속성
- 속성을 다른 언어로 설정해도 화면을 보여주는데는 지장이 없다.
- 검색 엔진에서 "특정 언어로 된 문서"로 검색 범위를 제한할 경우, 이 속성을 기준으로 검색한다.
- 스크린리더가 콘텐츠를 읽을 때 해당 언어에 적합한 발음을 제공한다.
