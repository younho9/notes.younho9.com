# {{ $frontmatter.title }}

- Web Storage API는 브라우저가 key/value 쌍을 [[쿠키(cookie)|쿠키]]보다 훨씬 직관적인 방식으로 저장할 수 있는 메커니즘을 제공한다.
- Web Storage에는 [[sessionStorage]] , [[localStorage]] 두 가지 메커니즘이 있다.
