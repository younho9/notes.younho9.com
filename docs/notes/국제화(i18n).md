- **International**은 **"국가들 간의"** 라는 의미
- 각 국의 언어나 표준에 맞게 콘텐츠를 제공하는 것.
  - 다국어 지원
  - 다국어 UI (언어가 바뀌더라도 UI가 망가져서는 안된다.)
  - 각국의 날짜, 시간, 화폐 단위에 맞게 제공한다.
