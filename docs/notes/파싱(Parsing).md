# {{ $frontmatter.title }}

- 파싱은 문장을 그것을 이루고 있는 구성 성분으로 분해하고, 그들 사이의 위계 관계를 분석하여 문장의 구조를 결정하는 것을 말한다.
- 컴퓨터 과학에서는 일련의 문자열을 의미있는 [[토큰(Token)]]으로 분해하고 이들로 이루어진 파스 트리를 만드는 과정을 말한다.
