# {{ $frontmatter.title }}

- 유저의 IP를 통해 접속 국가를 파악한다.
  - 접속 국가에 대한 정보를 [[HTTP-Request|HTTP Request]]에 실어서 서버에 전송한다.
  - 서버에서는 요청에 담긴 국가 정보를 통해 해당 접속 국가의 콘텐츠를 조회하여 전달한다.
