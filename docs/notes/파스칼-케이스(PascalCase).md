# {{ $frontmatter.title }}

- `PascalCase`
- 첫 단어를 대문자로 시작하는 표기법
- [[카멜-케이스(camelCase)|카멜 케이스(camelCase)]]에서 첫 글자만 대문자 변환하는 것으로 만들 수 있음.
- 예시: BackgroundColor, TypeName, PowerPoint
