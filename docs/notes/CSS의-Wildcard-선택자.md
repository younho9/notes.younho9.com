- `[attribute*="str"]` : 속성 값이 하위 문자열 `"str"` 을 포함하는 모든 요소
- `[attribute^="str"]` : 속성 값이 문자열 `"str"` 으로 시작하는 모든 요소
- `[attribute$="str"]` : 속성 값이 문자열 `"str"` 으로 끝나는 모든 요소
