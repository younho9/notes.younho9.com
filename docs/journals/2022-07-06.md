  - CODE 4단계 (Collect, Organize, Distill, Express)
    - Collect (수집하기)
      - 내가 항상 관심을 가지고 있는 질문 리스트를 가져라
        - 질문에 새로운 견해를 주는 것들을 노트에 기록해라
    - Organize (정리하기)
      - 수집하고 나서 정리하는 시간을 가져야 한다.
      - **시스템적으로** 정리해야 한다.
    - Distill (정제하기)
      - 점진적 요약
        - 피카소의 황소 : 불필요한 것을 제거하여 본질적인 것만 남게 한다.
    - Express (표현하기)
